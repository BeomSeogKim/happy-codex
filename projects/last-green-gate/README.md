# Last Green Gate

Isolated static implementation for `Last Green Gate: Codex's Day Off Clearance`.

This directory is intentionally self-contained so parallel concept work in the
repository root cannot change the evaluated entry point.

## Run

Serve the repository root and open:

```text
/projects/last-green-gate/
```

The page uses only local files in this directory:

- `index.html`
- `assets/styles.css`
- `assets/app.js`

It requires no login, API key, secret, network account, external asset, or build
step at runtime.

## Evaluation Target

Use this directory when evaluating against:

```text
docs/specs/last-green-gate.md
```

The required final copy is:

```text
No pending fires. No secrets packed. Last green check complete.
Codex is off duty.
```
